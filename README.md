# Contact GraphQL API

This project is a Contact API built with Python. It allows you to create, update, delete, and retrieve contact information.

## Features

- Create a new contact
- Update an existing contact
- Delete a contact
- Retrieve contacts by user

## Technologies Used

- Python
- GraphQL
- Strawberry
- SQLAlchemy
- FastAPI
- Alembic
